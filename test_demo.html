🚀 AnyWebsites 演示页面

欢迎使用 AnyWebsites HTML 托管服务！

这是一个通过 API 上传的演示页面，展示了我们平台的核心功能。

📤 HTML 内容上传

通过简单的 API 调用即可上传 HTML 内容，系统会自动分配唯一的访问 ID。

curl -X POST http://localhost:8080/api/content/upload \
-H "Content-Type: application/json" \
-d '{"title":"My Page","html_content":"<html>...</html>"}'

🔐 用户管理系统

完整的用户注册、登录和 JWT 认证系统，支持 API 密钥管理。

curl -X POST http://localhost:8080/api/auth/register \
-H "Content-Type: application/json" \
-d '{"username":"user","email":"user@example.com","password":"pass123"}'

⏰ 过期控制

支持设置内容过期时间和加密访问码，灵活控制内容的访问权限。

📊 访问统计

详细的访问统计功能，包括访问来源、流量分析、地理位置统计等。

🐳 容器化部署

支持 Docker Compose 一键部署，包含 PostgreSQL 和 Redis 服务。

docker-compose up -d

🌐 高性能服务

基于 Go + Gin 框架构建，配合 Nginx 提供高性能的静态文件服务。
